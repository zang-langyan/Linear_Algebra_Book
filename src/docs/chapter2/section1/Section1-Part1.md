# 第一节 线性方程组
## 一、线性方程组的定义

首先了解线性方程，包含形如 $x_{1}x_{2}$, $\sqrt{x_{1}x_{2}}$ ($x_{1}$ 和$x_{2}$ 是变量)的方程都不是线性方程，例如 $x_{1} + x_{2} = x_{1}x_{2}$，$x_{2} = 2\sqrt{x_1}$。

线性方程组则是由一个或几个包含相同变量 $x_{1}, x_{2}, \cdots, x_{n}$ 的线性方程组成的。

> **定义2.1.1:** 线性方程组是由若干个含有未知数的线性方程组成的方程组。其中每个方程的左边是一个线性多项式，右边是一定的常数。线性方程组的形式可以用矩阵运算来表示，即 $Ax = b$，其中 $A$ 为系数矩阵，$x$ 为未知向量，$b$ 为常数向量。

对于 $n$ 元线性方程组
$$
\begin{cases}
a_{11}x_{1} + a_{12}x_{21} + \cdots + a_{1n}x_{n} = b_{1}\\
a_{21}x_{1} + a_{22}x_{21} + \cdots + a_{2n}x_{n} = b_{2}\\
\cdots \cdots\\
a_{m1}x_{1} + a_{m2}x_{21} + \cdots + a_{mn}x_{n} = b_{m}
\end{cases}
$$
如果 $b_{i}(i = 1, 2, \cdots, m)$ 不全为零，则这个线性方程组称为 $n$ 元非齐次线性方程组，如果 $b_{1} = b_{2} = \cdots, b_{m} = 0$，形如
$$
\begin{cases}
a_{11}x_{1} + a_{12}x_{21} + \cdots + a_{1n}x_{n} = 0\\
a_{21}x_{1} + a_{22}x_{21} + \cdots + a_{2n}x_{n} = 0\\
\cdots \quad \cdots\\
a_{m1}x_{1} + a_{m2}x_{21} + \cdots + a_{mn}x_{n} = 0
\end{cases}
$$
的线性方程称为 $n$ 元齐次线性方程组。可以看出，齐次线性方程组一定有解 $x_{1} = x_{2} = \cdots = x_{n} = 0$，这个解为齐次线性方程组的零解。如果齐次线性方程组有唯一解，则这个唯一解必定时零解。当齐次线性方程组有无穷解时，称齐次线性方程组有非零解。

## 二、线性方程组的解
线性方程组的解有三种情况：
> - 无解：当系数矩阵与常数向量不能表示同一平面或同一直线时，该线性方程组无解；
> - 唯一解：当系数矩阵为非奇异矩阵时，线性方程组有且仅有唯一解；
> - 多解：当系数矩阵为奇异矩阵时，线性方程组存在无数个解。
>
> 需要指出的是：求包含两个变量的两个线性方程组的解，等价于求两条直线的交点；求包含三个变量的三个线性方程组的解，则其中每一个方程代表一个屏main，求解方程组相当于找到三个平面的交点。

## 三、矩阵的初等变换
矩阵的初等变换是一种重要的数学工具，它可以利用简单的操作改变矩阵的形式，从而解决一些数学方程，比如求解线性方程组和逆矩阵。

具体来说，矩阵的初等变换是指矩阵的元素可以通过一系列简单的操作进行变换，而不改变矩阵的表达式形式。矩阵的初等变换主要包括行变换、列变换和对角变换三种。

> **定义2.1.2:** 矩阵的下列三种变换称为矩阵的初等行变换：

- 交换矩阵的两行（交换 $i$，$j$ 两行，记作 $r_{i}\leftrightarrow r_{j}$）
- 以一个非零的 $k$ 乘矩阵的某一行（第 $i$ 行乘数 $k$，记作 $r_{i}\times k$
- 把矩阵的某一行的 $k$ 倍驾到另一行（第 $j$ 行乘数$k$ 加到第第 $i$ 行 ，记作 $r_{i}r_{j}\times k$）

把定义2.2.1中的“行”换成“”列，即得到矩阵的初等列变换定义（相应记号中把 $r$ 换成 $c$）。

初等行变换与初等列变换统称为 **初等变换**。需要指出的是，初等变换的拟变换仍是初等变换，且变换类型相同。

**例如**，变换 $r_{i}\leftrightarrow r_{j}$ 的逆变换即为本身；变换 $r_{i}\times k$ 的逆变换为 $r_{i}\times 1/k$；变换 $r_{i}+kr_{j}$ 的逆变换为 $r_{i}+(-k)r_{j}$ 或 $r_{i}-kr_{j}$。
